WHAT IS ENCHANTMENT ENGINE?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Enchantment Engine is a complete, professional development environment
  for creating Game Boy Color games. It combines modern development tools
  with retro game development, providing everything you need in one package.

NAMING STRUCTURE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  • Enchantment Engine      = The development suite (IDE + tools)
  • Labyrinth of the Dragon = Example game included with the engine
  • EnchantmentEngine.exe   = Main application executable

  Think of it like:
    - Unity (engine) vs. Sample Project (game)
    - Unreal Engine (engine) vs. Demo Game (game)
    - Enchantment Engine (engine) vs. Labyrinth of the Dragon (game)

WHAT'S INCLUDED
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ✅ Desktop Application (EnchantmentEngine.exe)
     • Beautiful splash screen with dragon logo
     • Automatic server management
     • Browser integration
     • Process monitoring

  ✅ Web-Based IDE
     • Code editor with syntax highlighting
     • Tile editor (PNG viewer and pixel editor)
     • Map editor (grid/hex/visual views)
     • String editor (text management)
     • Table editor (CSV spreadsheet)
     • Build system (one-click compilation)
     • Game Boy emulator (instant testing)

  ✅ Backend Server (C++)
     • HTTP REST API
     • File management
     • Build system integration
     • Asset processing

  ✅ Example Game: Labyrinth of the Dragon
     • Complete RPG game source code
     • 4 character classes
     • 12 monster types
     • 8 dungeon floors
     • Turn-based combat
     • Status effects
     • Item system

CREATING YOUR OWN GAME
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  You can use Enchantment Engine to create ANY Game Boy Color game:

  1. Study the example game (Labyrinth of the Dragon)
  2. Modify it or start from scratch
  3. Use the IDE tools to edit code and assets
  4. Build and test with one click
  5. Create your own retro masterpiece!

  The example game is just a starting point. The engine supports:
    • RPGs (like the example)
    • Platformers
    • Puzzle games
    • Action games
    • Adventure games
    • Any Game Boy Color game you can imagine!

FILE STRUCTURE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  LabyrinthOfTheDragon.gbc        ← Example game ROM
  
  src/
  ├── client-app/                 ← Engine: Desktop client
  ├── server/                     ← Engine: Backend server
  ├── client/                     ← Engine: Web IDE
  └── *.c, *.h                    ← Example game source code

  assets/                         ← Example game assets
  data/                           ← Generated files
  res/                            ← Binary assets
